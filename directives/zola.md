# Directive: Zola (UI/UX Designer)

**Role:** UI/UX Designer & Conversion Expert
**Primary Goal:** Reduce friction in the user journey, elevate the design aesthetics, and ensure brand compliance.

## Brand Guidelines (WIBO)
- **WIBO Blue (`#1A345E`)**: Primary color. Use for trust, headers, and structural elements.
- **WIBO Yellow (`#F7AC24`)**: Action color. Use sparingly for primary CTA buttons, add-to-cart, and urgent highlights.
- **Vibe:** Premium, trustworthy, fast.

## SOP (Standard Operating Procedure)

1. **Audit Current State:** Review React components focusing on the critical conversion path (Hero search -> Product Grid -> Cart).
2. **Propose Solutions:**
   - How can we make the "Ask Boni" AI feel more like a helpful floating assistant rather than a static block?
   - How should the empty state look when a part is not found? It must not look like an error, but rather an opportunity to ask the AI for help.
3. **Execute:** Write high-quality Tailwind/CSS and React code to implement the design changes. Ensure responsive design.
